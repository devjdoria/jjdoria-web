# JJDoria — Software Development Portfolio

Modern multi-language portfolio built with Next.js, showcasing web development, SaaS dashboards, landing pages and automation workflows.

---

## Live Website

https://www.devjjdoria.com

---

## Features

- Multi-language support (EN, ES, IT, CA, NL)
- Modern responsive UI
- Interactive demo previews
- Real contact form integration
- Professional domain email
- SEO optimization
- OpenGraph metadata
- Vercel Analytics
- Fast performance with Next.js 16

---

## Demo Projects

### CRM Dashboard

Conceptual SaaS dashboard for small businesses with analytics, metrics and client management.

### Restaurant Landing

Premium landing page for restaurants with modern visuals and responsive design.

### LeadFlow Automation

Automation workflow demo for lead capture and CRM/email integrations.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- next-intl
- Resend
- Vercel
- Framer Motion

---

## Local Development

Install dependencies:

```bash
pnpm install
```

Start development server:

```bash
pnpm dev
```

Open:

```txt
http://localhost:3000
```

---

## Deployment
